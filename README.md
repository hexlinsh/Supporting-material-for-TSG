# Supporting-material-for-TSG
Supplementary materials and datasets for the paper "An Intent-Driven LLM-based Agent Framework for Power System  Simulation Workflow"


This repository contains the supplementary materials, datasets, and execution logs for the manuscript:
"An Intent-Driven LLM-based Agent Framework for Power System Simulation Workflow" (Submitted to IEEE Transactions on Smart Grid).

Repository Structure
Supplementary_Material.pdf: Contains the Semantic Stress Matrix, Action Space definitions, System Prompt, and aggregated statistical results.
Detailed_Execution_Results.xlsx: The full results of the 450 ablation trials (Syntactic Compliance & Physical Execution Capability).
Execution_Logs.xlsx: The raw JSON execution logs and state-machine filtering records, providing transparency into the agent's reasoning process.

About the Data
The datasets provided here validate the framework's capability to bridge high-level semantic intents with deterministic physical simulation boundaries. For a quick overview of the framework's robustness across different complexity levels, please refer to the `Supplementary_Material.pdf`.
